¿Qué es una función Async en JavaScript?

async es una palabra clave que se usa para declarar una función asíncrona, es decir, una función que puede esperar operaciones que tardan en completarse, como llamadas a un servidor, lecturas de archivos o temporizadores.

Al declarar una función con async, esta devuelve automáticamente una promesa. Dentro de ella, se puede usar await para pausar la ejecución hasta que otra promesa se resuelva.
